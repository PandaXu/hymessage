# 🚀 快速运行项目指南

## ⚠️ 当前状态

检测到 **Xcode 尚未安装**，需要先安装 Xcode 才能运行 iOS 项目。

## 📋 运行前检查清单

- [ ] macOS 版本 ≥ 15.6（当前：15.1，需要升级）
- [ ] Xcode 已安装
- [ ] 项目文件完整

## 🎯 快速解决方案

### 步骤 1: 升级 macOS（如果还没升级）

```bash
# 检查可用更新
softwareupdate --list

# 升级到 macOS 15.7.2
sudo softwareupdate --install "macOS Sequoia 15.7.2-24G325"
```

或使用系统设置：
- 苹果菜单 → 系统设置 → 通用 → 软件更新

### 步骤 2: 安装 Xcode

**方法 A: 从 App Store 安装（推荐）**

```bash
# 打开 App Store
open -a "App Store"

# 然后搜索 "Xcode" 并安装
```

**方法 B: 从命令行安装（如果已安装 mas-cli）**

```bash
# 安装 mas-cli（如果还没有）
brew install mas

# 登录 App Store
mas signin your@email.com

# 安装 Xcode
mas install 497799835
```

### 步骤 3: 配置 Xcode

安装完成后：

```bash
# 接受许可协议
sudo xcodebuild -license accept

# 设置命令行工具路径
sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer

# 验证安装
xcodebuild -version
```

### 步骤 4: 运行项目

```bash
# 打开项目
cd /Users/heytea/HYMessage
open HYMessage.xcodeproj
```

然后在 Xcode 中：
1. 选择模拟器（如 iPhone 15 Pro）
2. 点击运行按钮（⌘R）

## 🔄 一键运行脚本

创建以下脚本可以自动完成上述步骤：

```bash
#!/bin/bash
# 检查并运行项目

echo "检查 Xcode 安装状态..."

if [ -d "/Applications/Xcode.app" ]; then
    echo "✅ Xcode 已安装"
    
    # 配置路径
    sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer
    
    # 打开项目
    cd /Users/heytea/HYMessage
    open HYMessage.xcodeproj
    
    echo "✅ 项目已打开，请在 Xcode 中选择模拟器并运行"
else
    echo "❌ Xcode 未安装"
    echo "请先安装 Xcode："
    echo "1. 打开 App Store"
    echo "2. 搜索并安装 Xcode"
    echo ""
    echo "或运行：open -a 'App Store'"
    open -a "App Store"
fi
```

## 🛠️ 替代方案（仅编译，无 UI）

如果暂时无法安装 Xcode，可以使用命令行工具验证代码：

```bash
# 检查 Swift 语法（不完整）
swiftc -typecheck HYMessage/HYMessageApp.swift 2>&1 | head -20
```

但**无法运行 iOS 应用**，因为需要：
- iOS Simulator（Xcode 提供）
- 完整的 iOS SDK
- 应用签名和打包工具
