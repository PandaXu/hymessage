# HYMessage - 短信智能管理App

一个功能强大的iOS短信智能管理应用，支持按短信签名分类和AI智能分类管理。

## 功能特性

### 1. 短信签名识别
- 自动识别短信中的签名信息（支持【】、[]、（）等多种格式）
- 按签名对短信进行分组展示
- 快速查找特定签名下的所有短信

### 2. AI智能分类
- 基于关键词和规则的智能分类算法
- 支持以下分类：
  - 验证码
  - 营销推广
  - 通知提醒
  - 金融理财
  - 物流快递
  - 社交娱乐
  - 工作相关
  - 其他

### 3. 分类管理
- 查看AI建议的分类
- 一键应用AI建议的分类
- 手动选择分类
- 按分类浏览短信

### 4. 搜索功能
- 支持按内容搜索
- 支持按发件人搜索
- 支持按分类筛选

## 技术栈

- **开发语言**: Swift 5.0+
- **UI框架**: SwiftUI
- **最低系统版本**: iOS 17.0
- **架构**: MVVM模式

## 使用说明

### 安装要求
- Xcode 15.0 或更高版本
- iOS 17.0 或更高版本的设备/模拟器

### 运行项目
1. 使用Xcode打开 `HYMessage.xcodeproj`
2. 选择目标设备或模拟器
3. 点击运行按钮（⌘R）

### 权限说明
应用需要以下权限：
- **短信权限**：用于读取和管理短信（需要在Info.plist中配置）
- **通讯录权限**：用于识别发件人（可选）

**注意**：由于iOS系统限制，实际设备上无法直接读取短信。当前版本使用模拟数据进行演示。如需在真实设备上使用，需要：
1. 使用MessageUI框架发送短信
2. 或通过第三方短信备份工具导入数据
3. 或使用iCloud同步功能
