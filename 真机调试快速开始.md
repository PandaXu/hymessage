# 🚀 真机调试快速开始

## 5 步快速上手

### 1️⃣ 连接设备
- 用 USB 线连接 iPhone/iPad 到 Mac
- 设备上点击"信任此电脑"
- 保持设备解锁

### 2️⃣ 打开项目
```bash
cd /Users/heytea/HYMessage
open HYMessage.xcodeproj
```

### 3️⃣ 配置签名
在 Xcode 中：
1. 点击左侧项目 "HYMessage"（蓝色图标）
2. 选择 Target "HYMessage"
3. 点击 "Signing & Capabilities" 标签
4. ✅ 勾选 "Automatically manage signing"
5. 选择你的 Team（Apple ID）

### 4️⃣ 选择设备
- 在 Xcode 顶部工具栏，点击设备选择器
- 选择你连接的 iPhone/iPad（不是模拟器）

### 5️⃣ 运行
- 点击运行按钮（▶️）或按 `⌘R`
- 首次运行需要在设备上信任开发者证书

## ⚡ 常见问题快速解决

| 问题 | 解决方法 |
|------|---------|
| 设备不显示 | 1. 重新插拔 USB<br>2. 设备解锁<br>3. 重启 Xcode |
| 需要开发者模式 | 设置 → 隐私与安全性 → 开发者模式 → 开启 |
| 未受信任的开发者 | 设置 → 通用 → VPN与设备管理 → 信任证书 |
| Bundle ID 冲突 | 修改为：`com.yourname.HYMessage` |

## 📱 设备要求

- iOS 17.0 或更高版本（项目要求）
- 已启用开发者模式（iOS 16+）
- 已信任此 Mac

## 💡 提示

- **付费账号**：无限制，可发布到 App Store
- **首次连接**：需要 USB 连接，之后可无线调试
